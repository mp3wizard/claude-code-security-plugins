# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this plugin, please report it responsibly:

1. **Do not open a public GitHub issue** for security vulnerabilities.
2. Use [GitHub's private vulnerability reporting](https://github.com/casedone/claude-code-security-plugins/security/advisories/new) to submit a report.
3. Include: description of the issue, steps to reproduce, and potential impact.

You can expect an acknowledgement within 72 hours and a resolution timeline within 14 days for confirmed issues.
